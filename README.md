# Super Math World: Number Kingdom

Y1 十位世界数学小游戏。玩家选择角色后，在横向跑跳关卡中收集金币、撞问号砖块回答数学题，并在 50 分后进入管道通关。

## Files

- `index.html` - 全部 HTML、CSS、JavaScript 游戏代码
- `sky.png` - 本地天空背景
- `pixel-quest-run.mp3` - 背景音乐

## How To Play

- 左右方向键：移动
- Space：跳跃
- 手机按钮：左、右、跳
- 撞问号砖块回答题目，答对加 10 分
- 碰到怪物会出现怪物挑战，答错才扣心
- 达到 50 分后进入管道通关

## GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `sky.png`, and `pixel-quest-run.mp3`.
3. Go to repository `Settings`.
4. Open `Pages`.
5. Set source to `Deploy from a branch`.
6. Choose `main` branch and `/root`.
7. Save, then open the GitHub Pages link.
